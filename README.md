# 树洞信件漂流站 (Tree Hole Letter Drift Station)

一个基于Android的匿名信件分享应用，让用户可以在树洞中倾诉，让信件随波漂流。

## 应用特色

- **轻仪式感 + 无压情绪羁绊**：提供轻松的写信体验，没有社交压力
- **树洞 + 木屋 + 漂流信**：温馨的UI设计，营造安全的倾诉环境
- **匿名信件分享**：用户可以匿名分享自己的情感和想法
- **信件漂流系统**：让你的信件被其他用户发现和回应

## 技术架构

- **平台**：Android
- **语言**：Kotlin
- **UI框架**：Jetpack Compose
- **架构**：MVVM + Repository Pattern
- **数据模型**：Letter, CollectionBox, DriftBottle

## 应用结构

### 主要功能模块

1. **信箱 (Home)**：查看收到的信件和通知
2. **探索 (Explore)**：发现其他用户分享的信件
3. **写信 (Create)**：创建和发送新的信件
4. **漂流瓶 (Connect)**：参与随机信件交换
5. **收藏 (Profile)**：管理个人收藏的信件

### 核心数据模型

- **Letter**：信件数据模型，包含发件人、收件人、内容和时间戳
- **CollectionBox**：信箱数据模型，用于组织和管理信件
- **DriftBottle**：漂流瓶数据模型，实现随机信件交换功能

## 主题设计

应用采用"树洞信件漂流站"主题，色调包括：

- **木屋棕** (#8D6E63)：主要UI元素
- **树洞绿** (#4CAF50)：强调色和按钮
- **邮票红** (#F44336)：通知和重要元素
- **纸张背景色** (#F5F5DC)：背景和卡片

### 构建应用

```bash

```

```bash
adb install app/build/outputs/apk/debug/app-debug.apk
```
